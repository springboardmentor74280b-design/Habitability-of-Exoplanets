4. WHY ACCURACY IS MISLEADING
--------------------------------------------------------------------------------
Class distribution in test set:
  Class 0:  799 (98.64%)
  Class 1:    4 ( 0.49%)
  Class 2:    7 ( 0.86%)

A 'dumb' classifier that predicts everything as class 0 would get:
  Accuracy: 98.64%
  Precision: 0.00 (no true positives)
  Recall: 0.00 (no true positives)
  F1 Score: 0.00

This is why we use F1, Precision, and Recall for imbalanced data!

5. KEY OBSERVATIONS
--------------------------------------------------------------------------------
✓ Best F1 Score: Linear SVM (0.997)
✓ Best Recall: Linear SVM (0.996)

Observations:
  - All models struggle with minority class (low recall)
  - class_weight='balanced' helps but is not sufficient
  - Severe class imbalance limits baseline performance
  - SMOTE and other sampling techniques are needed
